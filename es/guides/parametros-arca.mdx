# API de parámetros ARCA

ARCA mantiene tablas de referencia que se usan al emitir facturas: tipos de comprobante, alícuotas de IVA, países, monedas, tipos de documento, condiciones frente al IVA, unidades de medida y más.

Facturear expone estos parámetros a través de su API para que no tengas que hardcodearlos en tu sistema.

---

## ¿Por qué usar la API de parámetros?

- Los parámetros de ARCA **se actualizan periódicamente** (nuevos tipos de comprobante, cambios de alícuotas, nuevos países, etc.)
- Si hardcodeás los valores, tu sistema puede quedar desactualizado
- La API de Facturear siempre devuelve los valores vigentes directamente de ARCA

---

## Endpoints disponibles

### Listar todos los parámetros disponibles

```bash
GET /arca-parameters
```

Respuesta:
```json
{
  "parameters": [
    "Cbte_Tipo",
    "Concepto",
    "Doc_Tipo",
    "Iva_Tipo",
    "MConc_Tipo",
    "Mon_Tipo",
    "Opt_Compra",
    "Pais_Tipo",
    "PtoVta",
    "Tributos_Tipo",
    "UMed_Tipo"
  ]
}
```

### Obtener un parámetro específico

```bash
GET /arca/{name}
```


### Países (Pais_Tipo)

```bash
GET /arca/Pais_Tipo
```

Devuelve la lista completa de países con sus códigos para la Factura E.

### Monedas (Mon_Tipo)

```bash
GET /arca/Mon_Tipo
```

```json
[
  { "Id": "PES", "Desc": "Pesos Argentinos" },
  { "Id": "DOL", "Desc": "Dólar Estadounidense" },
  { "Id": "EUR", "Desc": "Euro" }
  // ...
]
```

### Tipos de documento (Doc_Tipo)

```bash
GET /arca/Doc_Tipo
```

Tipos de documento para identificar al receptor (DNI, CUIT, pasaporte, etc.)

## Ejemplo de uso en una integración

```javascript
// Al inicializar tu app, cacheá los parámetros
async function cargarParametrosARCA() {
  const [tiposComprobante, alicuotas, paises, monedas] = await Promise.all([
    fetch('https://api.facturear.com/arca/Cbte_Tipo', {
      headers: { 'x-api-key': API_KEY }
    }).then(r => r.json()),
    fetch('https://api.facturear.com/arca/Iva_Tipo', {
      headers: { 'x-api-key': API_KEY }
    }).then(r => r.json()),
    fetch('https://api.facturear.com/arca/Pais_Tipo', {
      headers: { 'x-api-key': API_KEY }
    }).then(r => r.json()),
    fetch('https://api.facturear.com/arca/Mon_Tipo', {
      headers: { 'x-api-key': API_KEY }
    }).then(r => r.json())
  ])
  
  // Guardá en caché (Redis, memoria, etc.)
  await cache.set('arca:tiposComprobante', tiposComprobante, 86400) // 24hs TTL
  await cache.set('arca:alicuotas', alicuotas, 86400)
  await cache.set('arca:paises', paises, 86400)
  await cache.set('arca:monedas', monedas, 86400)
}
```
