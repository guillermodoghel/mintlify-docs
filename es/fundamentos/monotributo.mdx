# Facturación electrónica para Monotributistas

Si sos **Monotributista** en Argentina, tenés la obligación de emitir facturas electrónicas desde 2019. Esta guía explica todo lo que necesitás saber: qué tipo de comprobante emitir, cómo configurarte y cómo usar Facturear para simplificar el proceso.

## ¿Qué tipo de factura emite un Monotributista?

Los Monotributistas **siempre emiten Factura C**, sin importar a quién le vendan:

| Tu cliente es... | Tipo de factura |
|---|---|
| Consumidor final | Factura C |
| Empresa / Responsable Inscripto | Factura C |
| Otra monotributista | Factura C |
| Organismo del Estado | Factura C |
| Cliente del exterior | Factura E (exportación) |

<Info>
  A diferencia de los Responsables Inscriptos que usan Factura A o B según el receptor, vos **siempre emitís Factura C**. La única excepción es la exportación de servicios al exterior (Factura E).
</Info>

## Configuración en ARCA (ex AFIP)

### Paso 1: Punto de venta para Monotributistas

En ARCA, los Monotributistas deben crear un punto de venta del tipo:

**"Factura Electrónica - Monotributo - Web Service"**

Este es diferente al punto de venta de los Responsables Inscriptos. Asegurate de elegir el correcto al configurar.

Ver la guía completa: [Crear un punto de venta en la guía de certificado digital](/es/guides/certificado-digital#paso-7-crear-un-punto-de-venta)

## Qué información va en la Factura C

Una Factura C de Monotributista debe incluir:

- **Datos del emisor**: nombre completo, CUIT, domicilio fiscal, condición frente a AFIP ("Monotributista")
- **Número de comprobante**: punto de venta (4 dígitos) + número correlativo (8 dígitos)
- **Fecha de emisión**
- **Datos del receptor**: según el monto
  - Hasta cierto umbral: "Consumidor Final" sin identificación
  - Montos mayores: nombre/razón social del receptor
- **Descripción del producto/servicio**
- **Importe total** (sin discriminar IVA)
- **CAE y fecha de vencimiento** (lo asigna ARCA automáticamente)
- **Código QR** (obligatorio desde 2021)

---

## ¿Cuándo identificar al receptor?

| Monto total | Identificación del receptor |
|---|---|
| Hasta $27.000 aprox. | "Consumidor Final" — sin datos del comprador |
| Más de $27.000 | Nombre y apellido o razón social |
| Cualquier monto si el cliente lo pide | CUIT/DNI + nombre |

Facturear maneja esto automáticamente según los datos que cargues.

## Límites y topes importantes

### Factura C sin identificar al receptor

Por resolución de ARCA, existe un **límite diario** para facturas a "Consumidor Final" sin identificar. Si superás ese límite en un día, las facturas subsiguientes deben identificar al receptor.

Facturear controla este límite automáticamente.

### Exportaciones de servicios

Si prestás servicios digitales al exterior (desarrollo, diseño, consultoría, etc.), podés emitir **Factura E** y acceder a beneficios como el cobro por divisas. Consultá con tu contador si esta opción te conviene.

## Recategorización y cambio de régimen

### Recategorización semestral

Los Monotributistas se recategorizan **dos veces al año** (enero y julio) en base a:
- Ingresos brutos de los últimos 12 meses
- Alquileres devengados
- Precio unitario máximo de venta

Facturear registra todas tus facturas con fecha y monto, lo que facilita el cálculo para la recategorización.

### Cuándo pasarse al Régimen General

Si superás el tope de categoría K del Monotributo o lo preferís por conveniencia fiscal, debés pasarte al **Régimen General** como Responsable Inscripto. Esto implica:

1. Darte de baja del Monotributo
2. Inscribirte en IVA (y Ganancias si corresponde)
3. Empezar a emitir Facturas A o B según el receptor
4. Actualizar la configuración en Facturear

## Integración via API

Si tenés un sistema propio (e-commerce, ERP, sistema de gestión), podés integrar Facturear via API REST:

```javascript
const response = await fetch('https://api.facturear.com/invoices', {
  method: 'POST',
  headers: {
    'x-api-key': 'tu_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    cuitId: 'tu-cuit-id',
    comprobante: {
      tipo: 11,  // Factura C
      puntoVenta: 1,
      receptor: {
        razonSocial: 'Consumidor Final',
        condicionIva: 'CF'
      },
      items: [
        {
          descripcion: 'Servicio de consultoría',
          cantidad: 1,
          precio: 50000
        }
      ]
    }
  })
})
```

Ver la [documentación completa de la API](/es/api-reference).

## Preguntas frecuentes

<AccordionGroup>
  <Accordion title="¿Puedo emitir Factura A siendo Monotributista?">
    No. Los Monotributistas **solo pueden emitir Facturas C** (o E para exportaciones). Si un cliente necesita Factura A para computar crédito fiscal de IVA, no podés dársela. Si esta situación es frecuente, considerá pasarte al Régimen General.
  </Accordion>
  <Accordion title="¿El Monotributo incluye IVA?">
    Sí. El Monotributo es un régimen simplificado que reemplaza tanto el IVA como el Impuesto a las Ganancias con una cuota fija mensual. Por eso la Factura C no discrimina IVA: está incluido en el precio total.
  </Accordion>
  <Accordion title="¿Tengo que emitir factura por todas las ventas?">
    Sí, estás obligado a emitir comprobante por cada operación. Si no emitís, ARCA puede sancionarte. Facturear facilita el proceso para que no quede ninguna venta sin documentar.
  </Accordion>
  <Accordion title="¿Qué pasa si no me recategorizé a tiempo?">
    Si no te recategorizás en el período correcto, ARCA puede hacerlo de oficio y puede implicar sanciones o intereses. Usá el historial de facturas de Facturear para calcular tus ingresos de los últimos 12 meses fácilmente.
  </Accordion>
  <Accordion title="¿Puedo facturar desde el celular?">
    Sí, Facturear es responsive y funciona desde cualquier dispositivo. También podés usar la API para integrar con cualquier app mobile.
  </Accordion>
</AccordionGroup>
