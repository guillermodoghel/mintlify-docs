# ⚡ Facturador Rápido

El Facturador Rápido es la forma más sencilla de crear facturas electrónicas en Facturear. Diseñado para facturas simples y ocasionales, te permite facturar en minutos con una interfaz intuitiva y minimalista.

## 🎯 ¿Qué es el Facturador Rápido?

El Facturador Rápido es una herramienta optimizada para crear facturas electrónicas de manera rápida y eficiente. Ideal para:

- ✅ Facturas simples y ocasionales
- ✅ Clientes individuales o pequeñas empresas
- ✅ Servicios básicos o productos simples
- ✅ Cuando necesitas facturar rápidamente

## 🚀 Características Principales

### ⚡ Proceso Rápido
- **3 pasos simples**: Datos del cliente → Productos/Servicios → Confirmar
- **Interfaz minimalista**: Solo los campos esenciales
- **Validación automática**: Verificación en tiempo real
- **Generación instantánea**: PDF listo en segundos

### 🎨 Interfaz Intuitiva
- **Diseño limpio**: Sin distracciones, enfocado en lo esencial
- **Autocompletado**: Sugerencias inteligentes para clientes frecuentes
- **Validación visual**: Indicadores claros de errores y éxito
- **Responsive**: Funciona perfectamente en móviles y tablets

### 🔧 Funcionalidades Incluidas
- **Datos del cliente**: CUIT, razón social, dirección
- **Productos/Servicios**: Descripción, cantidad, precio unitario
- **Cálculos automáticos**: Subtotal, impuestos, total
- **Generación de PDF**: Con código QR y validez fiscal

## 📋 Cómo Usar el Facturador Rápido

### Paso 1: Acceder al Facturador Rápido

1. **Inicia sesión** en tu cuenta de Facturear
2. **Navega** a "Facturación" en el menú lateral
3. **Selecciona** "Facturador Rápido"
4. **Elige tu CUIT** (si tienes múltiples)

### Paso 2: Completar Datos del Cliente

<Steps>
  <Step title="Información Básica">
    - **CUIT del cliente**: Se valida automáticamente con AFIP
    - **Razón social**: Se completa automáticamente
    - **Dirección**: Opcional, se puede completar manualmente
  </Step>
  <Step title="Validación">
    El sistema valida automáticamente:
    - ✅ Formato correcto del CUIT
    - ✅ Existencia en AFIP
    - ✅ Estado activo del contribuyente
  </Step>
</Steps>

### Paso 3: Agregar Productos/Servicios

**Para un producto/servicio:**

1. **Descripción**: Describe el producto o servicio
2. **Cantidad**: Número de unidades
3. **Precio unitario**: Precio por unidad
4. **IVA**: Se calcula automáticamente según configuración

**Para múltiples productos:**
- Haz clic en "Agregar otro producto"
- Repite el proceso para cada item
- El total se calcula automáticamente

### Paso 4: Revisar y Generar

1. **Revisa el resumen**: Verifica todos los datos
2. **Confirma la factura**: Haz clic en "Generar Factura"
3. **Descarga el PDF**: Disponible inmediatamente
4. **Envía al cliente**: Por email o descarga directa

## 💡 Consejos y Mejores Prácticas

### ✅ Datos del Cliente
- **CUIT válido**: Siempre verifica que el CUIT sea correcto
- **Razón social**: Confirma que coincida con AFIP
- **Dirección completa**: Incluye código postal para mejor validación

### ✅ Productos/Servicios
- **Descripción clara**: Usa descripciones específicas y detalladas
- **Precios correctos**: Verifica precios unitarios y cantidades
- **IVA apropiado**: El sistema aplica el IVA según tu configuración

### ✅ Proceso de Facturación
- **Revisa antes de generar**: Una vez generada, no se puede modificar
- **Guarda el PDF**: Descarga y guarda una copia local
- **Registra el número**: Anota el número de factura para seguimiento

## 🔧 Configuración Avanzada

### Valores por Defecto
Puedes configurar valores por defecto para agilizar el proceso:

1. **Ve a Configuración** → "Valores por Defecto"
2. **Establece valores comunes**:
   - Punto de venta por defecto
   - Tipo de comprobante
   - Condición de IVA
   - Moneda

### Clientes Frecuentes
Para clientes que facturas regularmente:

1. **Guarda información del cliente** en tu lista de contactos
2. **Usa autocompletado** para llenar datos automáticamente
3. **Crea plantillas** para productos/servicios comunes

## ❓ Solución de Problemas

### Errores Comunes

<error>
  **CUIT no encontrado** - Verifica que el CUIT sea correcto - Confirma que esté activo en AFIP - Revisa que no haya espacios o caracteres especiales
</error>

**Error de validación** - Revisa todos los campos obligatorios - Verifica que los precios sean números válidos - Confirma que las cantidades sean positivas

**Problema de generación** - Asegúrate de tener certificados activos - Verifica que tu CUIT esté configurado - Contacta soporte si persiste el problema
