# 📄 Factura Detallada

La Factura Detallada es la herramienta completa de Facturear para crear facturas electrónicas complejas. Ofrece control total sobre todos los campos, configuraciones avanzadas y es ideal para facturas empresariales que requieren precisión y detalle.

## 🎯 ¿Qué es la Factura Detallada?

La Factura Detallada es la funcionalidad más completa de Facturear, diseñada para:

- ✅ Facturas empresariales complejas
- ✅ Múltiples productos o servicios
- ✅ Configuración de tributos específicos
- ✅ Control total sobre todos los campos
- ✅ Facturas con configuraciones avanzadas

## 🚀 Características Principales

### 📋 Campos Completos
- **Datos del emisor**: Información completa de tu empresa
- **Datos del receptor**: Cliente con validación AFIP
- **Productos/Servicios**: Gestión detallada de items
- **Tributos**: Configuración de impuestos específicos
- **Observaciones**: Notas y comentarios personalizados

### ⚙️ Configuración Avanzada
- **Múltiples tributos**: IVA, impuestos provinciales, nacionales
- **Condiciones de pago**: Efectivo, transferencia, cheque, etc.
- **Monedas**: Pesos argentinos, dólares, euros
- **Puntos de venta**: Selección específica del POS
- **Tipos de comprobante**: A, B, C, según corresponda

### 🔧 Funcionalidades Avanzadas
- **Gestión de productos**: Base de datos de productos/servicios
- **Cálculos automáticos**: Subtotal, impuestos, totales
- **Validación en tiempo real**: Verificación de datos
- **Plantillas**: Guardar configuraciones frecuentes
- **Historial**: Acceso a facturas anteriores

## 📋 Cómo Usar la Factura Detallada

### Paso 1: Acceder a Factura Detallada

1. **Inicia sesión** en tu cuenta de Facturear
2. **Navega** a "Facturación" en el menú lateral
3. **Selecciona** "Factura Detallada"
4. **Elige tu CUIT** y punto de venta

### Paso 2: Configurar Datos del Comprobante

<Steps>
  <Step title="Información Básica">
    - **Tipo de comprobante**: A, B, C según corresponda
    - **Punto de venta**: Selecciona el POS apropiado
    - **Fecha de emisión**: Por defecto es la fecha actual
    - **Fecha de vencimiento**: Para pagos diferidos
  </Step>
  <Step title="Configuración Avanzada">
    - **Condición de pago**: Efectivo, transferencia, etc.
    - **Moneda**: ARS, USD, EUR
    - **Cotización**: Para monedas extranjeras
    - **Observaciones**: Notas adicionales
  </Step>
</Steps>

### Paso 3: Completar Datos del Cliente

**Información Obligatoria:**
- **CUIT**: Se valida automáticamente con AFIP
- **Razón social**: Se completa automáticamente
- **Domicilio**: Dirección completa del cliente
- **Condición de IVA**: Consumidor final, responsable inscripto, etc.

**Información Opcional:**
- **Email**: Para envío automático
- **Teléfono**: Datos de contacto
- **Código postal**: Para validación geográfica

### Paso 4: Agregar Productos/Servicios

**Para cada producto/servicio:**

1. **Código**: Código interno del producto
2. **Descripción**: Descripción detallada
3. **Cantidad**: Número de unidades
4. **Precio unitario**: Precio por unidad
5. **Descuento**: Porcentaje o monto fijo
6. **IVA**: Tipo de IVA aplicable
7. **Tributos adicionales**: Impuestos específicos

**Gestión de productos:**
- **Base de datos**: Guarda productos para reutilizar
- **Categorías**: Organiza por tipo de producto
- **Precios**: Mantén precios actualizados
- **Stock**: Control de inventario (opcional)

### Paso 5: Configurar Tributos

**Impuestos Nacionales:**
- **IVA**: 21%, 10.5%, 0% según corresponda
- **Impuestos internos**: Para productos específicos
- **Percepciones**: Según régimen aplicable

**Impuestos Provinciales:**
- **Ingresos Brutos**: Según jurisdicción
- **Impuestos municipales**: Si aplica
- **Otros tributos**: Según normativa local

### Paso 6: Revisar y Generar

1. **Revisa el resumen**: Verifica todos los datos
2. **Valida cálculos**: Confirma totales y subtotales
3. **Genera la factura**: Haz clic en "Generar Factura"
4. **Descarga PDF**: Con código QR y validez fiscal
5. **Envía al cliente**: Por email o descarga directa

## 🔧 Configuración Avanzada

### Plantillas de Facturación
Crea plantillas para agilizar el proceso:

1. **Ve a Configuración** → "Plantillas"
2. **Crea nueva plantilla**:
   - Nombre descriptivo
   - Configuración de tributos
   - Productos frecuentes
   - Datos por defecto

### Gestión de Productos
Organiza tu base de datos de productos:

1. **Categorías**: Organiza por tipo de producto
2. **Precios**: Mantén precios actualizados
3. **Códigos**: Sistema de codificación consistente
4. **Descripciones**: Detalladas y precisas

### Configuración de Tributos
Personaliza la configuración fiscal:

1. **Tributos por defecto**: Establece tributos comunes
2. **Cálculos automáticos**: Configura fórmulas personalizadas
3. **Validaciones**: Establece reglas de validación
4. **Reportes**: Configura reportes fiscales

## 📊 Casos de Uso Comunes

### 🏢 Factura Empresarial Completa
- **Múltiples productos**: Varios items con diferentes tributos
- **Descuentos**: Descuentos por volumen o cliente
- **Condiciones especiales**: Pagos diferidos o financiación
- **Observaciones**: Términos y condiciones específicos

### 🏭 Factura Industrial
- **Productos técnicos**: Especificaciones detalladas
- **Tributos específicos**: Impuestos internos o especiales
- **Certificaciones**: Cumplimiento de normativas
- **Trazabilidad**: Códigos de lote o serie

### 🛒 Factura Comercial
- **Gran cantidad de items**: Múltiples productos
- **Descuentos promocionales**: Ofertas y promociones
- **Programa de fidelidad**: Puntos o descuentos acumulados
- **Envío**: Costos de logística incluidos

## ❓ Solución de Problemas

### Errores Comunes

<error>
  **Error de validación AFIP** - Verifica que el CUIT sea correcto - Confirma que esté activo en AFIP - Revisa que no haya espacios o caracteres especiales
</error>

**Error en cálculos** - Verifica que los precios sean números válidos - Confirma que las cantidades sean positivas - Revisa la configuración de tributos

**Problema de generación** - Asegúrate de tener certificados activos - Verifica que tu CUIT esté configurado - Confirma que el punto de venta sea válido

### Validaciones Importantes
- **CUIT válido**: Formato correcto y activo en AFIP
- **Precios positivos**: Todos los valores deben ser positivos
- **Cantidades válidas**: Números enteros o decimales válidos
- **Tributos aplicables**: Verifica que los tributos sean correctos
